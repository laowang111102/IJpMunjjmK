# 社团管理系统平台 java55

## 项目概述

社团管理系统平台是一个基于SpringBoot和Vue的前后端分离项目，包含了详尽的文档。本项目旨在通过高效的信息管理，规范社团的日常运营，提高社团管理的效率。

## 技术栈

### 开发工具

- IntelliJ IDEA

### 后端框架

- SpringBoot
- MyBatis

### 前端框架

- Vue
- Element-UI
- Layui
- HTML
- CSS

### 运行环境

- JDK 1.8
- MySQL 5.7
- IntelliJ IDEA
- Maven 3
- Redis
- Node.js 14（建议版本12以上）

## 功能模块

本项目包括以下三个主要角色及其功能：

### 管理员角色

- 用户及社长信息管理
- 社团信息审核与维护
- 活动创建与管理
- 经费申请审批
- 新闻信息发布

### 社长角色

- 社团信息创建与管理
- 社团活动组织
- 新成员加入申请审核
- 发布社团新闻和通知
- 管理社团经费申请

### 用户角色

- 浏览社团信息
- 申请加入社团
- 参与社团活动报名
- 社团和活动评论及收藏
- 查看个人社团参与记录和活动报名状态

具体的目录结构将在文档中详细说明。

## 核心亮点

- 完善的权限管理，确保不同角色之间的有效协作和信息安全。
- 前后端分离的设计，提供更加灵活的系统部署和扩展能力。
- 详尽的文档支持，便于开发者和使用者理解和使用系统。
